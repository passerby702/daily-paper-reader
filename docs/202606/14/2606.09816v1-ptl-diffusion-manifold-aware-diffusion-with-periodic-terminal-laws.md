## 摘要
标准扩散模型通常使用单一时间齐次的高斯终端分布作为生成的参考定律。尽管这一选择在分析上方便且在经验上强大，但它对于集中在低维流形附近的数据几乎不提供显式结构，而数据分布的不同区域可能对应于不同的局部几何或语义因素。因此，反向模型必须几乎完全从无结构的终端参考分布中恢复流形级别的结构。我们提出了PTL-Diffusion，这是一种概念验证的扩散框架，其前向加噪过程收敛到一个非常数的周期高斯终端定律族，而不是单一的不变定律。与相位条件DDPM不同（其中相位信息仅进入去噪网络，而前向过程保持不变），PTL-Diffusion将相位结构直接嵌入前向加噪动力学中。所提出的构建与标准去噪扩散模型保持接近：对于周期受迫的Ornstein–Uhlenbeck型前向过程，我们推导了闭式的前向边缘分布、极限周期高斯终端族以及显式的高斯反向后验，从而实现了标准噪声预测训练。我们还引入了一个不变平均正则化项，通过平均周期参考定律耦合相位条件的反向动力学。在环面和圆柱点云基准以及Olivetti人脸数据集上的实验表明，PTL-Diffusion在匹配的DDPM基线之上改善了流形级别的分布匹配，减少了相位条件误差、特征空间协方差误差和最近邻流形距离。这些结果表明结构化终端参考定律是一个有希望的方向，同时激励更富表达力的相位构建和更大规模的评估。

## Abstract
Standard diffusion models typically use a single time-homogeneous Gaussian terminal distribution as the reference law for generation. While this choice is analytically convenient and empirically powerful, it provides little explicit structure for data concentrated near low-dimensional manifolds, where different regions of the data distribution may correspond to distinct local geometric or semantic factors. As a result, the reverse model must recover manifold-level structure almost entirely from an unstructured terminal reference distribution.   We propose PTL-Diffusion, a proof-of-concept diffusion framework whose forward noising process converges to a nonconstant periodic family of Gaussian terminal laws rather than to a single invariant law. Unlike a phase-conditioned DDPM, where phase information only enters the denoising network while the forward process remains unchanged, PTL-Diffusion embeds phase structure directly into the forward noising dynamics.   The proposed construction remains close to standard denoising diffusion models: for a periodically forced Ornstein--Uhlenbeck-type forward process, we derive closed-form forward marginals, the limiting periodic Gaussian terminal family, and explicit Gaussian reverse posteriors, enabling standard noise-prediction training. We also introduce an invariant-average regularization term coupling the phase-conditioned reverse dynamics through the averaged periodic reference law. Experiments on torus and cylinder point-cloud benchmarks and the Olivetti face dataset show that PTL-Diffusion improves manifold-level distributional matching over matched DDPM baselines, reducing phase-conditioned errors, feature-space covariance errors, and nearest-neighbour manifold distances. These results suggest structured terminal reference laws as a promising direction, while motivating more expressive phase constructions and larger-scale evaluations.

## 2. 方法论：核心思想、关键技术细节
- **核心思想**：构建前向加噪过程，使其收敛到一个**非常数的P-周期高斯终端分布族**，而非单一不变高斯分布。相位变量作为粗粒度坐标，组织数据变化，终端分布族提供结构化的参考。
- **关键技术细节**：
  - **周期性Ornstein–Uhlenbeck型前向过程**：
    \[
    x_{n+1} = \rho x_n + b_{r_0+n} + \sigma \varepsilon_{n+1}, \quad \varepsilon_n \sim \mathcal{N}(0, I_d)
    \]
    其中 \(b_n\) 是P-周期非恒定驱动力，满足 \(b_{n+P}=b_n\)，且非恒定（例如 \(b_n = a \sin(2\pi n/P) u\)）。
  - **闭式前向边缘分布**：\(q(x_n|x_0,r_0) = \mathcal{N}(m_{n,r_0}(x_0), \Sigma_n)\)，协方差 \(\Sigma_n = \sigma^2\frac{1-\rho^{2n}}{1-\rho^2} I_d\)，与标准OU过程相同。
  - **周期终端分布族**：
    \[
    \mu_r^* = \mathcal{N}\left(m_r^* = \sum_{j=0}^\infty \rho^j b_{r-1-j}, \,\, \Sigma^* = \frac{\sigma^2}{1-\rho^2} I_d\right)
    \]
    前向一步将 \(\mu_r^*\) 映射到 \(\mu_{r+1}^*\)，且任意初始分布经过足够步数后依Wasserstein-1距离收敛到对应相位的终端分布。
  - **高斯反向后验**：推导出闭式反向均值
    \[
    \tilde{\mu}_{n,r_0}(x_{n+1},\varepsilon) = \frac{1}{\rho}(x_{n+1} - b_{r_0+n}) - \kappa_n \varepsilon
    \]
    其中 \(\kappa_n = \frac{\sigma\sqrt{1-\rho^2}}{\rho\sqrt{1-\rho^{2(n+1)}}}\)，从而支持标准噪声预测训练。
  - **相位条件反向模型**：使用正弦/余弦相位嵌入 \(\phi(r) = (\sin\frac{2\pi r}{P}, \cos\frac{2\pi r}{P})\)，网络预测噪声 \(\varepsilon_\theta(x_n, n, \phi(r_n))\)。
  - **不变平均正则化**：基于提升态空间（相位×状态）上的不变测度 \(\Pi^* = \frac{1}{P}\sum_r \delta_r \otimes \mu_r^*\)，提出损失
    \[
    \mathcal{L}_{\text{avg}} = \mathbb{E}\left[ \left\| \frac{1}{P}\sum_{s=0}^{P-1} \varepsilon_\theta(x_n, n, \phi(s)) - \varepsilon \right\|^2 \right]
    \]
    耦合不同相位下的去噪器，确保它们共享一致的均值结构。
- **算法流程**：
  - **前向**：从训练数据 \((x_0, r_0)\) 出发，直接采样 \(x_k = \rho^k x_0 + a_{k,r_0} + \sqrt{1-\rho^{2k}}\varepsilon\)。
  - **反向采样**：从终端近似分布 \(x_K \sim \mathcal{N}(a_{K,r_0}, (1-\rho^{2K})I)\) 开始，逐相位逆向。
- **数据自适应驱动力构造**：对于相位标签 \(r_i\)，计算相位中心 \(c_r\)，中心化后定义 \(b_r = \tilde{c}_{r+1} - \rho \tilde{c}_r\)，确保零均值条件，满足理论要求。

## 6. 主要结论与发现
- PTL-Diffusion在环面和圆柱数据集上相比DDPM和Phase-DDPM**显著降低了边缘分布误差、流形约束误差、相位条件误差和不变平均误差**。例如环面 \(E_{\text{phase}}\) 从0.240（DDPM）降至0.057，\(E_{\text{avg}}\) 从0.159降至0.039。
