## 摘要
潜在扩散模型（LDMs）在视觉合成中占据主导地位，但其质量-计算权衡很大程度上受限于分词器的固定压缩率。可变长度分词器（VLTs）通过改变分词数量实现自适应压缩，使扩散模型能够灵活平衡质量和计算。然而，传统的VLTs通过截断有序分词序列来调节长度，这使得分词语义依赖于分词位置，并破坏了跨长度的表示对齐。这导致潜在分布中出现跨长度偏移，阻碍了单个可变长度扩散模型有效运行。为解决此问题，我们提出了一种新颖的可变长度分词器，通过合并分词来调节长度。我们表明，当扩散变换器按照合并模式运行时，鼓励相似的分词合并能够实现直接的跨长度表示对齐。由于传统的合并方法依赖于数据，使得生成过程中无法访问合并模式，我们引入了可学习的全局合并，该方法是数据无关的，以确保与扩散变换器的兼容性。在ImageNet 256×256生成任务上，我们基于合并的可变长度分词器与扩散变换器集成，相比先前的VLT方法，实现了更优越的gFID-计算权衡。代码可在[此链接](https://github.com/movinghoon/lgm)获取。

## Abstract
Latent Diffusion Models (LDMs) have become dominant in visual synthesis, but their quality-compute trade-off is largely constrained by the tokenizer's fixed compression ratio. Variable-length tokenizers (VLTs) promise adaptive compression by varying token counts, allowing diffusion models to flexibly balance quality and compute. However, conventional VLTs modulate length by truncating ordered token sequences, which makes token semantics depend on token position and breaks representational alignment across lengths. This leads to a cross-length shift in the latent distribution that hinders a single variable-length diffusion model from operating effectively. To address this, we propose a novel variable-length tokenizer that modulates length by merging tokens. We show that encouraging similar tokens to merge enables direct cross-length representation alignment when the diffusion transformer operates according to the merging pattern. Since conventional merging methods are data-dependent, making the merging pattern inaccessible during generation, we introduce learnable global merging, which is data-independent, to ensure compatibility with diffusion transformers. On ImageNet 256$\times$256 generation, our merging-based variable-length tokenizer integrated with a diffusion transformer achieves a superior gFID-compute trade-off compared to prior VLT methods. # 论文总结：Variable-Length Tokenization via Learnable Global Merging for Diffusion Transformers

## 1. 核心问题与整体含义（研究动机和背景）

- **核心问题**：现有的潜在扩散模型（LDMs）依赖固定压缩率的分词器，无法在单个模型中灵活地平衡生成质量与计算成本。可变长度分词器（VLTs）通过调整 token 数量承诺实现自适应压缩，但传统 VLTs（如嵌套 dropout）通过截断有序 token 序列来调节长度，导致 token 语义依赖于其位置（早期 token 编码高层语义，后期 token 编码低层细节）。这种截断方式破坏了**跨长度的表示对齐**（representational alignment），即不同 token 长度下各数据点在潜在空间中的相似性结构不一致，使得潜在分布及其对应的得分函数随 token 长度剧烈变化，从而阻碍了单个可变长度扩散模型在不同 token 数量下有效工作。
- **整体含义**：要真正实现扩散模型灵活的质量-计算权衡，需要一种能保持跨长度表示一致性的可变长度分词机制。

## 2. 方法论

- **核心思想**：通过**合并（merging）** 而非截断来调节 token 长度。当扩散模型按照合并模式（即哪些 token 被合并）运行时，合并操作可以定义出“全长度等价”的潜在表示（通过投影矩阵 W），使得原本不同长度的潜在特征能在相同维度下进行直接对齐。通过鼓励合并相似的 token，可以最小化表示偏移（公式 5），从而促进跨长度的表示对齐。

- **关键技术细节**：
    1.  **合并式长度调节**：给定一个分配矩阵 Γ，将 N 个原始 token 合并为 N-K 个合并 token（公式 2）。为处理合并 token 的位置信息，使用**比例注意力**（proportional attention）来考虑每个合并 token 的有效大小（即它聚合了多少原始 token）。这使得对合并 token 的处理等价于对全长度等价潜在表示 Wz 的标准注意力处理。
    2.  **可学习全局合并（Learnable Global Merging, LGM）**：为解决合并模式在生成时不可用的问题，提出数据无关的合并策略。引入一组可学习的全局嵌入 e ∈ R^{N×D}，对于每个减少量 K，通过凝聚聚类（agglomerative clustering）基于 e 之间的余弦相似度产生硬分配 Γ。为优化离散的聚类操作，采用**直通估计器（straight-through trick）**，并结合软分配 Γ_soft 来传播梯度。
    3.  **对齐损失（Alignment Loss）**：为了引导可学习嵌入 e 合并相似的潜在 token z，引入对齐损失 L_align，迫使 token-wise 的相似性（z_i 与 z_j 的余弦相似度）与对应的嵌入相似性（e_i 与 e_j 的余弦相似度）一致。训练总损失为 L_total + λ_align L_align。
    4.  **应用于扩散变压器**：训练单一扩散变压器 (DiT) 在所有长度的合并潜在表示上（通过随机采样 K），并在 DiT 中使用比例注意力和合并后的位置嵌入（˜pe = Γ̄^⊺ pe）。后处理阶段可为每个 token 长度引入轻量级 LoRA 参数进行微调。

- **结论**：基于可学习全局合并（LGM）的可变长度分词器能有效促进跨长度表示对齐，显著优于基于嵌套 dropout 的传统 VLT。

## 7. 优点

- **方法创新性**：首次将“合并”范式引入可变长度分词器，并创造性地用可学习全局嵌入解决生成时的合并模式缺失问题，同时通过直通估计器实现端到端训练。
- **本质解决核心问题**：通过合并实现跨长度潜在表示的**直接对齐**（公式 5），从根本上解决了截断法导致的表示漂移问题，原理清晰。
- **训练效率高**：无需多个独立模型，仅需训练一个扩散模型，后处理 LoRA 开销极小，参数增加可控。
- **实验设计规范**：在固定骨干和训练设置下与嵌套 dropout 对比，控制变量严谨；消融实验覆盖了所有关键组件。
