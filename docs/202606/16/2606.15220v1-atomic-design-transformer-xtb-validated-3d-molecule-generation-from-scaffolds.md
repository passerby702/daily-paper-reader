## 摘要
我们提出了一种SE(3)不变的变换器，用于三维分子生成，即原子设计变换器（ADT）。ADT每次自回归地放置一个原子。SE(3)不变性通过分词化实现：每个新原子的位置被编码在先前放置原子的局部坐标系中。网络是一个简单的因果变换器。为了评估，我们引入了xTB验证率（XVR），它检查在xTB GFN2几何弛豫后分子拓扑是否保持不变。在QM9上，ADT与最先进的基线方法具有竞争力。对于GEOM-Drugs，我们提出了一个支架条件三维生成的定量基准，在截断至30个原子的基态GEOM-Drugs数据集上评估了七个支架（苯、吡啶、嘧啶、吡嗪、呋喃、噻吩、环己烷）。XVR范围从11.1%（吡嗪）到29.7%（苯），全部来自单个训练模型。生成的分子化学多样性高，并且由于简单的因果变换器骨干架构，生成速度快。应用于截断至50个原子的GEOM-Drugs数据集时，XVR较低，而每键和每角的质量得以保持。这些结果使ADT成为计算机辅助分子设计的实用提出者。

## Abstract
We present an SE(3)-invariant transformer for 3D-molecule generation, the Atomic Design Transformer (ADT). ADT places atoms one at a time, autoregressively. The SE(3) invariance is achieved by tokenization: each new atom's position is encoded in a local coordinate frame of a previously placed atom. The network is a plain causal transformer. For evaluation, we introduce xTB-Validated Rate (XVR), which checks whether the molecular topology is preserved after xTB GFN2 geometry relaxation. On QM9, ADT is competitive with state-of-the-art baselines. For GEOM-Drugs, we present a quantitative benchmark of scaffold-conditional 3D generation, evaluating seven scaffolds (benzene, pyridine, pyrimidine, pyrazine, furan, thiophene, cyclohexane) on the 30-atom-truncated ground-state GEOM-Drugs dataset. XVR ranges from 11.1% (pyrazine) to 29.7% (benzene), all from a single trained model. The generated molecules are chemically diverse, and generation is fast by virtue of the plain causal-transformer backbone. Application to the 50-atom-truncated GEOM-Drugs dataset yields lower XVR while the per-bond and per-angle quality is retained. These results position ADT as a practical proposer for in silico molecular design.

# 原子设计变换器：基于支架的 xTB 验证的三维分子生成——论文详细总结

## 1. 核心问题与整体含义（研究动机和背景）

3D 分子生成是计算化学与药物设计的关键任务。现有方法主要分为两类：  
- **扩散模型**（如 EDM、GeoLDM、MiDi）需要多轮去噪，计算成本高且难以发挥标准 Transformer 的工具链优势。  
- **自回归模型**（如 G‑SchNet、Quetzal、InertialAR）通常采用等变图神经网络或混合架构（Transformer 生成隐变量 + 额外扩散/高斯头输出位置），**没有任何工作证明纯 Transformer 可以直接处理 3D 分子生成**。

此外，**支架条件生成**（给定一个固定结构基元，补全完整分子）是药物设计中先导优化、片段生长、支架跃迁的核心任务，但已有研究仅停留在定性演示，**缺乏定量基准**。同时，评估生成分子是否真正在量子力学水平上稳定（拓扑不因松弛而变化）也缺少实用指标。

**本文目标**：提出完全离散的纯 Transformer 自回归模型 ADT，首次实现 **SE(3) 不变性与条件生成分离**（不变性仅通过输入编码实现），并建立首个 **支架条件 3D 生成定量基准**，引入物理有效性指标 XVR。

## 2. 方法论：核心思想、关键技术细节与流程

### 2.1 核心思想：局部坐标系 + 全离散 Tokenization
- 每次放置一个新原子时，将其位置**表达在父原子的局部坐标系**中，而非全局坐标系。  
- 通过 Gram‑Schmidt 从两个祖先边（父→子，祖父→父）构建正交帧 `{e1, e2, e3}`。  
- 这样，全局旋转/平移不会改变帧内相对坐标，**SE(3) 不变性由构造保证**，网络只需普通因果 Transformer。

### 2.2 Token 设计
每个非终结原子事件由 **7 个离散 Token** 组成：  
| 槽位 | 含义 |
|------|------|
| `a` | 动作：INIT, CHAIN, ANGLE, ADD, LINK, END |
| `f` | 偏移指针：指向父原子（相对偏移，范围截断到 50） |
| `Z` | 原子序数 |
| `rb` | 键距：200 个对数间距 bin（0.80–2.50 Å） |
| `h0,h1,h2` | HEALPix 方向编码（Nside=16，3072 个像素），用于球面方向 |

对于 bootstrap 三原子（第 0–2 步），使用特殊 ANGLE 动作固定初始帧。LINK 动作用于成环，不新增原子，但提供一致性约束。

### 2.3 网络架构与训练
- **模型**：12 层因果 Transformer，隐藏维度 768，参数量 85.7 M，无位置编码（空间信息已隐含在 Token 序列中）。
- **训练**：随机前沿自由顺序增强。每个 epoch 对每个分子随机选择根原子和放置顺序，使模型等效地看到多种支架前缀，实现无重训练的支架条件生成。
- **损失**：全部 Token 的交叉熵（包括距离 bin 作为分类问题，无回归损失）。

### 2.4 生成流程
- 种子前缀（支架）被 tokenize 后作为输入，模型逐个 token 自回归生成，直到产生 END token。  
- 温度 T=1.0 采样，生成速度约 **20 ms/原子**（单 RTX 4090）。

## 3. 实验设计

### 3.1 使用的数据集
- **QM9**：134k 小分子（≤9 个重原子），B3LYP 优化单构象，用于无条件生成校验。
- **GEOM-Drugs**：303 k 个类药分子，每个取 GFN2‑xTB 优化最低能构象。按重原子数截断至 ≤30（85%）和 ≤50（99.9%），用于支架条件生成。

### 3.2 评价指标与基准
- **mol_stable**：RDKit 价键检查，经典指标。
- **XVR（xTB‑Validated Rate）**：新提出指标，管道：  
  1. 距离感知 + 芳构化预处理 → 加氢 → 生成 `pre‑SMILES`。  
  2. GFN2‑xTB 几何优化 → 弛豫后重感知 → `post‑SMILES`。  
  3. 若 `pre == post`（拓扑保持），则该分子通过 XVR。  
- **对比方法**：EDM、MiDi、GeoLDM（扩散）；Quetzal、InertialAR（混合自回归）。

### 3.3 实验设置
- **QM9**：无条件生成 10 000 个分子（从训练集帧缓存中采样 bootstrap 三元组）。
- **GEOM-Drugs 30‑atom**：7 个支架（苯、吡啶、嘧啶、吡嗪、呋喃、噻吩、环己烷），每支架生成 10 000 个分子。
- **GEOM-Drugs 50‑atom**：同 7 支架，从 30‑atom 模型微调后生成 10 000 个/支架。
- **结构精度分析**：对 XVR‑positive 分子统计键长/键角在弛豫前后的偏差（图6）。
- **失败模式分解**：将 `kekulized + xTB‑OK` 的子集分为 A3（完全保持）、A2（连接保持但键序不同）、A1（连接改变），并定性分析 C‑S/N‑S 键断裂等模式（附录 E）。

- **充分性与公平性**：
  - 对比了当前主流的扩散和混合自回归方法，QM9 上 mol_stable 相当（87.5% vs 89.4% GeoLDM, 90.4% Quetzal），但未对比 XVR（其他方法未报告）；Drugs 上因其他方法缺少定量支架条件数据，横向比较受限。

## 6. 主要结论与发现

1. **纯 Transformer 可以胜任 3D 分子生成**：ADT 在 QM9 上达到 mol_stable 87.5%，XVR 85.7%，接近但略低于最佳混合/扩散模型，但架构更简单。
2. **支架条件生成首次定量最优**：单个模型处理 7 种骨架，30‑atom 上 XVR 范围 11.1%–29.7%（均值 17.7%）；50‑atom 上 XVR 降至 4.8%–17.6%（均值 9.6%）。
3. **局部几何精度极高**：键长中位误差 ~0.01 Å，键角 ~2.3°，与 X 射线晶体学精度相当，且与分子尺寸无关。
4. **生成化学多样性强**：XVR‑positive 分子中 Bemis–Murcko 骨架独特率约 63%–86%，中位 Tanimoto 相似度 ~0.11（与 ChEMBL 随机对相当）。
5. **XVR 下降主要是长程拓扑错误**（环闭合、时序结束令牌定位不准），而非局部结构恶化。

## 7. 优点

- **架构极简**：完全无需等变层或扩散头，SE(3) 不变性由 Tokenization 保证，所有标准 Transformer 工具链（KV 缓存、批推理、温度采样、前缀条件等）直接可用。
- **训练高效**：单阶段交叉熵损失，不涉及连续优化或混合目标。
- **生成快速**：~20 ms/原子，明显快于扩散模型（数百步迭代）。
- **物理验证指标 XVR**：首次将 xTB 量子力学弛豫纳入评估，比纯 RDKit 检查更严格。
- **随机前沿训练**：使模型隐式学习大量骨架上下文，实现零额外训练的支架条件生成。
- **结果可复现**：代码、权重、评估工具均已开源。

## 8. 不足与局限

- **支架条件 XVR 偏低**（平均 ~17.7%），远不如无条件 QM9（85.7%），说明类药大分子条件生成仍极具挑战；50‑atom 更低（~9.6%）。模型在大分子上的长程位置错误和过早结束是主要原因。
- **分子尺寸限制**：offset 硬截断为 50，虽然够用但非层次化，可能限制更大分子生成。
- **未建模构象系综**：训练仅使用单构象（GFN2‑xTB 最低能），忽略多构象灵活性，可能导致生成的分子在真实环境中不稳定。
- **失败模式集中**：C‑S、N‑S 键易断裂（A1 类别占 5.5%–11.4%）；某些稠环芳构化感知不稳定（A2 约 <1%）。磷化学尚未学习。
- **未与 SBDD 方法对比**：论文提到未来可扩展到口袋条件生成，但目前缺少与 TargetDiff、DiffSBDD 等口袋感知模型在结合位点上的对比。
- **消融不充分**：未测试不同方向离散方案、不同距离分箱数量、不同 offset 截断等超参数影响。
- **硬件资源信息不够细**：未报告批大小、显存占用等。
