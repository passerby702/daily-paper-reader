## 摘要
流模型和扩散模型在多种模态中生成高质量样本，然而由于底层微分方程的数值积分，推理过程中需要大量网络评估。流映射通过直接学习微分方程的解映射来缓解这一问题，从而实现少步采样。然而，当前方法仅限于近似常微分方程的解映射。这些方法可用于学习随机微分方程的转移核，从而获得一个恢复过程边际分布（弱收敛）而非解路径（强收敛）的解映射。我们提出强随机流映射作为学习加性噪声随机微分方程强解映射的新框架，直接将确定性流映射推广到随机环境。此外，引入布朗运动的多项式近似，并证明其路径收敛。这些结果为扩散模型的解映射提供了无模拟训练目标。我们证明强随机流映射在图像生成上优于先前随机流映射方法，并能够实现分子系统的少步采样。

## Abstract
Flow and diffusion models generate high-quality samples in many modalities; however, many network evaluations are required during inference due to numerical integration of an underlying differential equation. Flow maps alleviate this problem by learning the solution map of the differential equation directly, enabling few-step sampling. Yet, current methods are restricted to approximating the solution map of ODEs. These methods can be used to learn the transition kernel of an SDE, thereby obtaining a solution map that recovers the marginal distributions of the process (weak convergence) rather than the solution path (strong convergence). We propose Strong Stochastic Flow Maps (SSFMs) as a novel framework for learning the strong solution map of additive-noise SDEs, directly generalizing deterministic flow maps to the stochastic setting. Further, a polynomial approximation to Brownian motion is introduced and shown to converge pathwise. These results enable a simulation-free training objective for the solution map of diffusion models. We demonstrate that SSFMs outperform previous stochastic flow map methods on image generation and enable few-step sampling of molecular systems.

### 1. 论文的核心问题与整体含义（研究动机和背景）

- **背景**：流模型和扩散模型在图像、视频、蛋白质设计、玻尔兹曼采样等多模态生成任务中取得了最先进性能。然而，这些模型依赖神经微分方程，推理时需要对底层微分方程进行数值积分，导致需要大量网络评估（NFEs）。为提升采样效率，近期研究提出学习**流映射（flow map）**，即直接学习微分方程的解映射，从而支持少步采样。但现有流映射方法仅限于学习**常微分方程（ODE）** 的解映射，无法直接适用于扩散模型的随机微分方程（SDE）过程。
- **核心问题**：现有方法（如一致性模型、Mean Flows）虽能加速采样，但只能近似ODE解；即使有工作尝试将流映射拓展到随机场景，它们学习的是SDE的**转移核（transition kernel）**，即仅恢复边际分布（弱收敛），而不是路径解（强收敛）。这意味着这些方法无法估计路径依赖的可观测值（如沿特定噪声路径的生成结果）。
- **本文目标**：提出**强随机流映射（Strong Stochastic Flow Maps, SSFMs）**，一种学习**加性噪声SDE的强解映射**的新框架，将确定性流映射直接推广到随机设置。核心是学习**Itô映射**，即给定初始条件和布朗运动实现，输出SDE路径解。

### 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程

#### 核心思想
SSFMs学习加性噪声SDE的**Itô映射**：  
`Ψ_{s,t}(X_s, W_{[s,t]}) → X_t`  
其中 `W_{[s,t]}` 是布朗运动路径。该映射满足两个条件：
- **切线条件**（tangent condition）：当 s→t 时，映射的增量等于SDE的漂移和扩散。
- **半群条件**（semigroup property）：Ψ_{s,t} = Ψ_{u,t} ∘ Ψ_{s,u}（路径一致性）。

#### 关键技术细节
1. **参数化**：采用类Euler-Maruyama形式：  
   `Ψ_{s,t}(X_s, W_{[s,t]}) = X_s + f_{s,t}(X_s, W_{[s,t]})(t-s) + g_{s,t}(W_{[s,t]})(W_t - W_s)`  
   其中 `f_{s,t}` 和 `g_{s,t}` 分别表示归一化的漂移积分和扩散积分，由神经网络参数化。
2. **布朗运动逼近**：为高效传递路径信息，采用**分段Legendre多项式展开**对布朗运动进行近似。定义系数  
   `I^{(n)}_{s,t} = ∫_s^t Ẽ_n((u-s)/(t-s)) dW_u`  
   性质：系数独立正态分布、均值为0、方差(t-s)/(2n+1)；且直接支持**Chen关系**（半群条件的闭合形式）。多项式近似随N增加在α-Hölder距离下收敛到真实布朗运动。
3. **自蒸馏训练目标**：通过最小化两个损失项联合学习：
   - **匹配损失** L_{f,g}：迫使 `f_{t,t}(X_t)=f(t,X_t)`，`g_{t,t}=g(t)`，即小步长内近似真实SDE系数。
   - **蒸馏损失** L_D：强制半群条件成立，即多步组合与单步预测一致。
4. **算法流程**（Algorithm 1）：
   - 采样时间对 (s,t)，其中 t = s + Δt 或 t > s+Δt。
   - 对匹配目标：使用标准Euler-Maruyama步作为目标（可视为加权系数匹配）。
   - 对蒸馏目标：计算两步预测（s→u→t）与单步预测（s→t）的均方误差，用EMA参数更新。
   - 训练中需访问真实漂移和扩散系数，对于扩散模型，这些系数可从条件得分推导得出，因此训练是**无模拟**的。

#### 关键公式
- 自蒸馏损失：`L_SD(v̂) = L_{f,g}(v̂) + L_D(v̂)`
- L_{f,g}：期望下的漂移/扩散系数匹配，L_D：半群一致性。
- 布朗多项式系数Chen关系：  
  `I^{(n)}_{s,t} = Σ_{m=0}^{n} c_{n,m} [ I^{(m)}_{s,u} + (-1)^{n+m} I^{(m)}_{u,t} ]`
