## 摘要
在文本到图像（T2I）生成中注入创造力是一个重大挑战，因为它要求合成图像不仅展现视觉新颖性和意外性，还要具有艺术价值。然而，当前的T2I模型大多针对字面上的文本-图像对齐及其数据分布进行优化，其噪声预测网络将生成限制在高概率区域，因此产生的输出缺乏真正的创造力。为解决这个问题，我们提出了一种自创造扩散（SCDiff）模型，用于有意义的T2I生成，该模型具有两个核心模块：可学习的空间加权（LSW）模块和视觉-语义混合损失（VSML）。LSW模块设计了一个参数化的Kaiser-Bessel窗口来增强中心图像特征，从而促进新颖且令人意外的生成。VSML模块引入了一个双重损失函数：相似性损失约束新图像与其文本描述一致，而多样性损失最大化其与原始图像的差异，从而增强语义价值和视觉新颖性。大量实验表明，我们的模型显著提高了创造力、语义对齐和视觉连贯性，为生成创造性对象提供了一个简单而强大的框架。

## Abstract
Instilling creativity in text-to-image (T2I) generation presents a significant challenge, as it requires synthesized images to exhibit not only visual novelty and surprise, but also artistic value. Current T2I models, however, are largely optimized for literal text-image alignment with their data distribution, and their noise prediction networks constrain the generation to high-probability regions, consequently generating outputs that lack authentic creativity. To address this, we propose a Self-Creative Diffusion (SCDiff) model for meaningful T2I generations featuring two core modules: a learnable spatial weighting (LSW) module and a visual-semantic mixing loss (VSML). The LSW module designs a parametric Kaiser-Bessel window to reinforce central image features, fostering novel and surprising generation. The VSML module introduces a dual loss function: a similarity loss constrains that the new images align with its textual description, while a diversity loss maximizes its distinction from the original image, enhancing both semantic value and visual novelty. Extensive experiments demonstrate that our model substantially improves creativity, semantic alignment, and visual coherence, offering a simple yet powerful framework for generating creative objects.

### 2. 论文提出的方法论
论文提出了一个名为 **SCDiff (Self-Creative Diffusion)** 的模型，其核心思想是通过空间域的特征调制和语义-视觉双重约束，来诱导扩散模型生成创意内容。模型包含两个核心模块：

*   **可学习的空间加权（Learnable Spatial Weighting, LSW）模块**:
    *   **核心思想**：直接在被U-Net编码器的浅层特征图上进行空间调制，通过增强图像中心区域的特征来打破模型的“均值回归”倾向，激发结构上的新颖性。
    *   **关键技术细节**：
        *   采用一个参数化的**Kaiser-Bessel窗口**`w(β)`，替代传统频域编辑（如C3方法）或简单的圆形/高斯窗口。该窗口具有紧支撑特性（在半径`R`外为零），避免了频域硬截断产生的全局振铃/鬼影伪影。
        *   调制操作定义为一个混合公式：`x_KB = x · (1 - w(β)) + α · x · w(β)`，其中`α`是可学习的放大因子，`β`控制窗口形状。
        *   窗口函数公式：当像素到中心的距离`r ≤ R`时，`w(β) ∝ I0(β√(1 - (r/R)²))`，否则为0。`I0`是零阶第一类修正贝塞尔函数。这种设计能在最大化中心能量集中的同时，保证边缘平滑过渡到零，使解码器更倾向于进行全局结构重组，而非局部纹理变化。
*   **视觉-语义混合损失（Visual-Semantic Mixing Loss, VSML）模块**:
    *   **核心思想**：作为评价器，联合优化LSW模块的参数`α`和`β`，以平衡视觉新颖性、意外性和语义价值。
    *   **关键技术细节**：
        *   **双重损失设计**：
            *   **多样性损失** (最大化): `1 - Simg(I_org, I(α, β))`，鼓励生成的图像`I(α, β)`与原始图像`I_org`在特征上有所差异，以提升新颖性。
            *   **相似性损失** (最大化): `S_text(T_p, I(α, β))`，约束生成图像与文本提示`T_p`的语义对齐，确保其有意义性。
        *   **优化策略**：采用分层优化方法。
            1.  **优化`α`**：固定`β`，使用**贝叶斯优化（Bayesian Optimization）** 高效寻找能使无约束VSML目标最大化的最优放大因子`α*`（因为`α`的评估昂贵，需要一次完整的扩散去噪过程）。
            2.  **优化`β`**：在`α*`基础上，使用**同步扰动随机逼近（SPSA）**方法求解带有约束（保证图像与原始图像的相似度高于阈值`τ`，防止图像严重失真）的自变量优化问题，以微调窗口形状`β`。
